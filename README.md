# Calculator Flutter App

A simple calculator application built with Flutter.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Clear and delete operations
* Decimal calculations

## Tech Stack

* Flutter
* Dart


## How to Run

Clone the repository:

```bash
git clone https://github.com/m-hassanali-tech/calculator-flutter.git
```
